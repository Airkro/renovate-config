# renovate-config

Shareable renovate config presets

## Usage

```jsonc
// .github/renovate.json
{
  "extends": ["github>airkro/renovate-config"]
}
```

or

```jsonc
// renovate.json
{
  "extends": ["local>{owner}/renovate-config"]
}
```
